SimpleLoading 组件演示

🎯 组件特点

  • ✅ 完全独立，不依赖任何主题配置
  • ✅ 简单易用，零配置即可使用
  • ✅ 响应式设计，支持移动端
  • ✅ 可自定义文本和进度
  • ✅ 平滑动画效果

🚀 快速使用

在浏览器控制台中运行以下命令：

显示加载页面：

window.showSimpleLoading()

隐藏加载页面：

window.hideSimpleLoading()

📝 组件使用方法

在Vue组件中使用：

<template> <SimpleLoading :visible="showLoading" text="正在加载数据..." :show-progress="true" :progress="loadingProgress" @loaded="handleLoadingComplete" /> </template> <script setup> import SimpleLoading from './components/SimpleLoading.vue' import { ref } from 'vue' const showLoading = ref(false) const loadingProgress = ref(0) const handleLoadingComplete = () => { showLoading.value = false } // 显示加载 const startLoading = () => { showLoading.value = true } </script>

⚙️ 组件属性

属性名 类型 默认值 说明
visible Boolean true 是否显示加载页面
text String '加载中...' 加载提示文本
showProgress Boolean false 是否显示进度条
progress Number 0 加载进度 (0-100)

🎨 自定义样式

组件使用了独立的CSS样式，不会受到主题变化影响。如需自定义样式，可以修改 SimpleLoading.vue 文件中的样式部分。

💡 使用场景

  • 页面初始化加载
  • 数据请求等待
  • 文件上传进度
  • 表单提交处理
  • 路由切换过渡
